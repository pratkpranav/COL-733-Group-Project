What?

A tool to keep track of your portfolio in real-time. Use the Visualizer in order to get the live chart of the asset, and use the portfolio tracker in order to keep track of the current value of your portfolio.

How?

We display the quotes downloaded from Yahoo Finance, using the Python API. Along with that, we use several frameworks that aid in distributed computing and storage such as Kafka, Cassandra, Redis, and Spark. We used Flask and Node.JS for our frontend.
